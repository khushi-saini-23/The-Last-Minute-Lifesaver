# 🌻 The Last-Minute Lifesaver

An AI-powered, cozy, and deeply empathetic study nook designed to help anxious students and developers clear their mental clutter, break the task-freeze, and organize their "mind-dumps" into bite-sized, actionable steps. 

Built with **React, TypeScript, Express**, and powered by the **Gemini 3.5 Flash** model.

> 🛠️ **AI Studio Deployment Note:** View your app setup directly in AI Studio via [this link](https://ai.studio/apps/5c770da9-3eed-4999-9193-b0483d56420c).

---

## 🐾 Meet Your Companion: Biscuit

When deadlines approach, panic shouldn't take over. **The Last-Minute Lifesaver** provides a warm, distraction-free environment featuring:
* **🧠 Cozy Mind-Dump Analyzer:** Drop your unorganized, messy thoughts or stressful task lists. The Gemini API structures them seamlessly into clear priorities without any judgment.
* **🐱 Biscuit the Cat (Chatbot):** A tiny, animated orange cat living in your study nook who responds with soft meows, purrs, and warm encouragement to help you breathe through anxiety.
* **🍃 Daily Reflection Journal:** Mood-based prompt generation to ground your thoughts and cultivate mindfulness.
* **🌦️ Offline Emergency Fallback:** If the API network takes a breather, your cozy companion activates a built-in local fallback path so your support system never goes down.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite), TypeScript, Tailwind CSS
* **Backend:** Node.js, Express.js
* **AI Integration:** Google GenAI SDK (`gemini-3.5-flash`)
* **Environment Management:** Dotenv
